## What is Microsoft Azure

### Core Idea

* Microsoft Azure is a **cloud platform** with a large set of services
* Enables building, deploying, and managing applications on a **global infrastructure**

👉 Key idea:

* Azure = **everything you need in the cloud**

---

### What Azure Offers

* **Global scale** → services available worldwide
* **Flexibility** → use your preferred tools & frameworks
* **Integrated platform** → combine infrastructure, data, AI, and analytics
* **Security & trust** → built-in security and compliance

👉 Key idea:

* One platform for **all IT needs**

---

### What You Can Do with Azure

* Run applications (e.g. Virtual Machines)
* Store and manage data
* Build modern applications (web apps, APIs)
* Use advanced services:

  * AI & Machine Learning
  * IoT
  * Analytics

👉 Key idea:

* From **simple hosting → to advanced cloud solutions**

---

## Azure Service Categories (Overview)

| Category                | Description                         | Examples                                           |
| ----------------------- | ----------------------------------- | -------------------------------------------------- |
| **Compute**             | Run applications and workloads      | Virtual Machines, App Services, Azure Functions    |
| **Networking**          | Connect and secure resources        | Virtual Network (VNet), Load Balancer, VPN Gateway |
| **Storage**             | Store data in different formats     | Blob Storage, Azure Files, Disk Storage            |
| **Databases**           | Managed database services           | Azure SQL Database, Cosmos DB                      |
| **AI + ML**             | Build intelligent applications      | Azure AI Services, Azure Machine Learning          |
| **Identity + Security** | Manage access and protect resources | Azure Active Directory (Entra ID), Key Vault       |
| **DevOps + Management** | Monitor and manage resources        | Azure Monitor, Azure DevOps                        |
| **Analytics**           | Process and analyze data            | Synapse Analytics, Data Factory                    |
| **IoT**                 | Connect and manage devices          | IoT Hub                                            |
| **Integration**         | Connect apps and services           | Logic Apps, Service Bus                            |

---

### Cloud Evolution

* Start: move existing apps to VMs (**lift-and-shift**)
* Then:

  * use managed databases
  * enable autoscaling
  * build event-driven applications

👉 Key idea:

* Cloud is more than just "running servers"

---

### Practical Example

* App with fluctuating demand:

  * scale up → handle peak load
  * scale down → reduce costs

👉 Benefit:

* Pay only for what you use
