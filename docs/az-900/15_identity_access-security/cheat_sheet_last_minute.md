# ⚡ AZ-900 Last-Minute Cheat Sheet (1-Minute Review)

## 🔐 Identity & Access
- Entra ID = identity system
- Tenant = identity boundary
- RBAC = who can do what
- Conditional Access = IF → THEN rules

---

## 🔑 Authentication
- Password → weak  
- MFA → strong  
- Passwordless → best  

👉 Use MFA or Passwordless

---

## 🔄 External Identities
- B2B → partners (guest users)
- Direct Connect → tenant-to-tenant
- B2C → customers

---

## 🛡️ Security Models
- Zero Trust → never trust
- Defense-in-Depth → layered security

---

## 🔒 Data Protection
- Encryption:
  - At rest
  - In transit
- Key Vault:
  - Secrets
  - Keys
  - Certificates

---

## 🛡️ Threat Protection
- Defender for Cloud:
  - Assess
  - Secure
  - Defend
- Secure Score = security level

---

## 🧠 Ultra Memory

- RBAC → access control  
- Conditional Access → conditions  
- MFA → multiple factors  
- Passwordless → best  
- B2B vs B2C → partners vs customers  
- Encryption → everywhere  
- Zero Trust → verify always

## 🚀 Final Sentence
