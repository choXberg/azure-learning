# AZ-900 Exam Traps & Patterns

## ⚠️ 1. "Least management" = Serverless

👉 Keywords:
- least effort
- no infrastructure
- minimal admin

✅ Answer → Azure Functions

---

## ⚠️ 2. "Lift and shift" = VM

👉 Keywords:
- migrate existing app
- minimal changes

✅ Answer → Virtual Machines

---

## ⚠️ 3. Web App without OS = App Service

👉 Keywords:
- web app
- no OS management

✅ Answer → App Service

---

## ⚠️ 4. Containers: ACI vs AKS

- Simple → ACI  
- Complex / orchestrated → AKS  

---

## ⚠️ 5. Blob vs SQL

- Files → Blob  
- Structured data → SQL  

---

## ⚠️ 6. Scalability vs Elasticity

- Scalability → manual or planned  
- Elasticity → automatic  

---

## ⚠️ 7. High Availability vs Fault Tolerance

- High Availability → minimal downtime  
- Fault Tolerance → zero interruption  

---

## ⚠️ 8. CapEx vs OpEx

- CapEx → upfront  
- OpEx → pay-as-you-go  

---

## ⚠️ 9. RBAC vs NSG

- RBAC → permissions (who can access)  
- NSG → traffic (network filtering)  

---

## ⚠️ 10. Monitoring Tools

- Azure Monitor → metrics/logs  
- Application Insights → app-level  
- Log Analytics → query logs  

---

## ⚠️ 11. Pricing traps

- Reserved → cheapest long-term  
- Spot → cheapest but risky  
- Pay-as-you-go → flexible  

---

## ⚠️ 12. Shared Responsibility

👉 Azure:
- hardware
- datacenter

👉 You:
- data
- access
- configuration  

---

## ⚠️ 13. Keywords matter!

Watch for:
- "best"
- "most cost-effective"
- "least effort"

👉 They change the answer!

---

## 🧠 Final Tip

If unsure:

1. Eliminate wrong answers
2. Look for keywords
3. Choose:
   - simplest solution
   - least management
