# ☁️ Cloud Concepts (AZ-900)

This module covers the **core cloud principles** you need for the AZ-900 exam.
It explains what cloud computing is, how responsibility is shared, and how pricing and deployment models work.

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain what cloud computing means
- compare public, private, and hybrid cloud models
- understand the shared responsibility model
- describe the consumption-based pricing approach
- recognize common cloud pricing principles and benefits

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **CapEx vs. OpEx**
- **public vs. private vs. hybrid cloud**
- **shared customer/provider responsibilities**
- **pay-as-you-go pricing**
- **scalability and elasticity basics**

---

## 🚀 Study Tip

For AZ-900, focus on the **business meaning and use case** of each concept rather than deep technical implementation.
