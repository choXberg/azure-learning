# 🛠️ Tools and Deployment (AZ-900)

This section introduces important Azure tools used to **interact with, manage, and deploy resources**.
It also covers Azure Arc and infrastructure automation concepts that are relevant for the AZ-900 exam.

---

## 🎯 Topics Covered

- [Tools for Interacting with Azure](10_tools_for_interacting_with_azure.md)
- [Azure Arc](20_azure_arc.md)
- [Azure Resource Manager and ARM Templates](30_resourcemanager_and_Azure_arm_templates.md)

---

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- compare the **Azure Portal**, **Azure CLI**, and **Azure PowerShell**
- understand the purpose of **Azure Arc**
- explain what **Azure Resource Manager (ARM)** does
- recognize when **templates** help automate deployments

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **Portal** for graphical management
- **CLI / PowerShell** for automation and scripting
- **ARM templates** for repeatable deployments
- **Azure Arc** for managing hybrid and multi-cloud resources

---

## 🚀 Study Tip

In AZ-900, these topics are usually tested at a **high conceptual level**.
You should know **what each tool is for** and **when it is commonly used**.
