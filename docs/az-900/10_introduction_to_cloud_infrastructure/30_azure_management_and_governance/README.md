## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain how Azure helps manage and optimize costs
- understand the purpose of **tags**, **Azure Policy**, and **resource locks**
- distinguish between **Advisor**, **Service Health**, and **Azure Monitor**
- recognize governance and compliance tools used in Azure

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **pricing and cost control tools**
- **governance with Azure Policy and resource locks**
- **monitoring vs. recommendations vs. service status**
- **compliance and trust-related services**
