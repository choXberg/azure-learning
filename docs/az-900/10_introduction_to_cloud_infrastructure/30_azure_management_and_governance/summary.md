# 📊 Azure Management and Governance – Summary (AZ-900)

## 💰 Cost Factors

- Resource type (VM, storage, etc.)
- Consumption (usage-based pricing)
- Geography (region impacts cost)
- Network traffic (egress costs money)
- Subscription type (free credits, limits)
- Marketplace (third-party costs)
