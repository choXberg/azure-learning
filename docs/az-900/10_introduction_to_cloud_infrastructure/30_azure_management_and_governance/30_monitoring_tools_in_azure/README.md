# 📈 Monitoring Tools in Azure (AZ-900)

This section introduces the main Azure tools used to **monitor services, receive recommendations, and stay informed about platform health**.

---

## 🎯 Topics Covered

- [Azure Advisor](10_azure_advisor.md)
- [Azure Service Health](20_azure_service_health.md)
- [Azure Monitor](30_azure_monitor.md)

---

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain the purpose of **Azure Advisor**
- understand what **Azure Service Health** reports
- describe how **Azure Monitor** collects and analyzes telemetry
- distinguish between recommendations, health status, and monitoring data

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **Advisor = recommendations**
- **Service Health = Azure service and outage information**
- **Azure Monitor = telemetry, metrics, logs, and alerts**
