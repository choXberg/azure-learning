# 💰 Azure Pricing Calculator (AZ-900)

## 🧠 Overview
- The **Azure Pricing Calculator** is a tool to estimate potential Azure costs
- Available online
- Used for:
  - Cost estimation before deployment
  - Budget planning

👉 Important:
- It is **only an estimate**
- ❌ No resources are created
- ❌ No charges are incurred

---

## ⚙️ What Can You Estimate?

- Compute (e.g., Virtual Machines, App Services)
- Storage (type, tier, redundancy)
- Networking (data transfer, bandwidth)

👉 You can estimate:
- Single resources
- Full solutions
- Example scenarios

---

## 🧩 Key Features

- Configure services and settings
- Compare:
  - Regions
  - Pricing tiers
  - Redundancy options
- Adjust configurations dynamically

---

## 🧪 Practical Example

👉 Scenario: Planning a web application

You can model:
- App Service plan
- Managed database
- Storage

👉 Then compare:
- Monthly cost across regions
- Different service tiers
- Redundancy options
