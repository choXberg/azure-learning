# Microsoft Purview – Summary

## Overview
Microsoft Purview is a **data governance, risk, and compliance solution family** that provides a **unified view of your data estate** across:

- On-premises environments
- Multicloud environments
- Software-as-a-Service (SaaS) platforms

It helps organizations understand, manage, and protect their data.

---

## Core Capabilities

Microsoft Purview enables:

- **Automated data discovery**  
  Automatically scans and identifies data across environments.

- **Sensitive data classification**  
  Detects and labels sensitive information (e.g., personal or financial data).

- **End-to-end data lineage**  
  Tracks how data moves and transforms across systems.

---

## Solution Areas

Microsoft Purview consists of two main areas:

### 1. Risk and Compliance

Integrates closely with Microsoft 365 services like:

- Microsoft Teams  
- OneDrive  
- Exchange  

**Key benefits:**

- Protect sensitive data across **clouds, apps, and devices**
- Identify data risks and ensure **regulatory compliance**
- Support compliance initiatives and audits

---

### 2. Unified Data Governance

Provides centralized governance across diverse data sources, including:

- Azure services  
- SQL and Hive databases  
- On-premises systems  
- Other clouds (e.g., Amazon S3)

**Key benefits:**

- Create a **complete data map** with classification and lineage
- Identify where **sensitive data is stored**
- Enable secure **data discovery for users**
- Generate insights on **data usage and storage**
- Manage **data access securely at scale**

---

## Key Takeaways (Exam Focus)

- Microsoft Purview = **Unified data governance + risk & compliance**
- Works across **on-prem, Azure, multicloud, SaaS**
- Provides:
  - Data discovery  
  - Data classification  
  - Data lineage  
- Helps with:
  - **Data protection**
  - **Compliance requirements**
  - **Data visibility and control**
