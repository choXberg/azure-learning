# Azure Advisor

## What is Azure Advisor?

**Azure Advisor** is a built-in Azure service that analyzes your deployed resources and provides **personalized recommendations** to improve your environment.

👉 Think of it as a **best-practices engine** for your Azure setup.

Each recommendation includes actions that you can:
- Apply immediately
- Postpone
- Dismiss

You can also configure **notifications** to be alerted when new recommendations are available.

---

## Key Capabilities

- Evaluates your Azure resources continuously
- Provides actionable recommendations
- Helps optimize:
  - reliability
  - security
  - performance
  - cost
  - operations

---

## Advisor Dashboard

The Azure Advisor dashboard gives a centralized view of all recommendations across your environment.

### You can filter by:
- Subscription
- Resource group
- Service

---

## Recommendation Categories

Azure Advisor organizes recommendations into five main categories:

### 🔵 Reliability
- Helps ensure applications stay available
- Identifies configuration risks

### 🔐 Security
- Detects vulnerabilities and potential threats
- Suggests improvements to reduce attack surface

### ⚡ Performance
- Recommends optimizations to improve speed and efficiency

### ⚙️ Operational Excellence
- Suggests improvements in deployment and workflow processes

### 💰 Cost
- Identifies opportunities to reduce Azure spending

---

## Benefits

- Improves overall system health
- Provides **proactive guidance**
- Helps reduce costs and risks
- Supports continuous optimization
