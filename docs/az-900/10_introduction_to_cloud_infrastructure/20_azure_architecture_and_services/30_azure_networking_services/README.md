# 🌐 Azure Networking Services (AZ-900)

This section introduces the **core Azure networking services** you need to know for the AZ-900 exam.
It focuses on connectivity, name resolution, secure communication, and network protection.

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain what an **Azure Virtual Network (VNet)** is
- distinguish between **VPN** and **ExpressRoute**
- understand the purpose of **Azure DNS**
- describe how **Network Security Groups (NSGs)** control traffic
- recognize common Azure networking and security scenarios

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **VNet = private network in Azure**
- **VPN = encrypted connection over the internet**
- **ExpressRoute = private dedicated connection**
- **DNS = name resolution**
- **NSGs = allow/deny inbound and outbound traffic rules**

---

## 🚀 Study Tip

For AZ-900, focus on the **purpose and differences** of each networking service rather than deep configuration details.

A common exam pattern is:

1. identify the connection type needed
2. decide whether it should be **internet-based** or **private dedicated**
3. choose the matching Azure networking service
