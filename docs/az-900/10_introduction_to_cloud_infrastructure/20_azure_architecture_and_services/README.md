## 📚 What You Should Know for the Exam

After completing this module, you should be able to:

- describe Azure’s global infrastructure and organizational hierarchy
- identify common compute, networking, and storage services
- compare service types based on use case and management level
- understand basic connectivity, security, and redundancy concepts
- recognize which Azure service best fits a business scenario

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **Regions, availability zones, and resource groups**
- **VMs vs. containers vs. serverless services**
- **Virtual networks, VPN, and ExpressRoute**
- **Storage types and redundancy options**
- **Common Azure service use cases**

---

## 🚀 Study Tip

A good AZ-900 strategy is to focus less on deep configuration details and more on:

1. **what the service does**
2. **when to use it**
3. **how it differs from similar services**
