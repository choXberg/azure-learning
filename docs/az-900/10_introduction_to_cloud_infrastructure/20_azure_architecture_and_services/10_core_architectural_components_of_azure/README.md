# 🏛️ Core Architectural Components of Azure (AZ-900)

This section covers the **fundamental building blocks of Azure**.
It explains how Microsoft Azure is organized physically and logically, and how resources are structured for management, billing, and governance.

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain what Microsoft Azure is
- understand Azure’s **global infrastructure**
- distinguish between **regions**, **availability zones**, and **region pairs**
- describe the relationship between **resources**, **resource groups**, **subscriptions**, and **management groups**

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **Azure regions and availability zones**
- **resource groups** as logical containers
- **subscriptions** for billing and access boundaries
- **management groups** for governance across subscriptions

---

## 🚀 Study Tip

AZ-900 often tests whether you can place an Azure concept in the right layer of the hierarchy.
Think from **smallest to largest**: resource → resource group → subscription → management group.
