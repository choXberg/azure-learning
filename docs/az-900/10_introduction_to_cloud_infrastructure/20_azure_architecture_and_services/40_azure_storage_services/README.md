# 💾 Azure Storage Services (AZ-900)

This section covers the **main Azure storage services** and storage-related concepts that are relevant for the AZ-900 exam.
It helps you understand where data is stored, how redundancy works, and which storage option fits a given scenario.

## 📚 What You Should Know for the Exam

After this section, you should be able to:

- explain the purpose of an **Azure Storage Account**
- distinguish between **Blob, File, Queue, and Table storage**
- understand **redundancy options** such as LRS, ZRS, GRS, and GZRS
- identify the right storage service for common business scenarios
- recognize basic Azure migration and file transfer options

---

## 🧠 Exam Focus Areas

Pay special attention to:

- **Blob Storage** for unstructured data
- **Azure Files** for shared file storage
- **Queue Storage** for messaging between components
- **Table Storage** for NoSQL key-value data
- **redundancy and replication options** for availability and durability

---

## 🚀 Study Tip

For AZ-900, you do not need deep implementation details.
Focus on these three questions:

1. **What kind of data is being stored?**
2. **How available or durable must it be?**
3. **Which Azure storage service is the best fit?**
