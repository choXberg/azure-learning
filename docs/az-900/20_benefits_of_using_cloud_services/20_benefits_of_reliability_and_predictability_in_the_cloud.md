## Reliability & Predictability (Cloud Benefits)

### Reliability

* Ability of a system to **recover from failures** and continue running
* Based on **decentralized cloud design**
* Resources can run in **multiple regions worldwide**

👉 Key idea:

* If one region fails → another region continues

👉 Supported by:

* automatic failover
* global infrastructure

👉 Exam tip:

* Reliability = **resilience + failover**

---

### Predictability

* Ability to **forecast performance and costs**
* Based on best practices (Well-Architected Framework)

---

### Performance Predictability

* Ensures consistent performance for users

👉 Supported by:

* autoscaling → adjusts resources automatically
* load balancing → distributes traffic
* high availability → avoids downtime

👉 Key idea:

* System adapts automatically to demand

---

### Cost Predictability

* Ability to **estimate and control cloud costs**

👉 Supported by:

* real-time monitoring
* usage tracking
* analytics & trends

👉 Tools:

* Azure Pricing Calculator

👉 Key idea:

* You can **forecast and optimize spending**

---

## 🧠 Quick Memory Guide

* Reliability → **recover from failures**
* Predictability → **know performance & costs**
* Performance → autoscale + load balance
* Cost → monitor + estimate

---

## 🎯 Exam Triggers

👉 "System continues after failure"
→ Reliability

👉 "Distribute traffic across resources"
→ Load balancing (performance predictability)

👉 "Automatically adjust resources"
→ Autoscaling

👉 "Estimate future costs"
→ Cost predictability / Pricing Calculator
