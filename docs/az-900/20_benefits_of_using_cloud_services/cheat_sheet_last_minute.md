# AZ-900 Cheat Sheet – Cloud Benefits (Last Minute)

## ☁️ Core Benefits

* High Availability → services stay online
* Scalability → add/remove resources
* Elasticity → automatic scaling
* Reliability → consistent performance

---

## 🔁 Predictability

* Performance predictable
* Cost predictable (tools available)

---

## 🛡️ Security & Governance

* Built-in security tools
* Compliance support
* Policies and controls

---

## ⚙️ Manageability

* Manage via:

  * Portal
  * CLI
  * PowerShell
* Automation possible

---

## 🌱 Sustainability

* Optimized resource usage
* Energy-efficient datacenters

---

## ⚠️ Exam Must-Know

* Scalability ≠ Elasticity
* High availability ≠ Reliability
* Cloud = operational efficiency

---

## 🎯 Final Tip

Think in benefits:

> Why is cloud better than on-prem?
